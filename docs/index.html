Gambling Discourse Analyzer

Measures gambling-culture vocabulary in Chinese retail-investor texts. Browser demo of cn-gambling-discourse. Only disposition-marking vocabulary counts towards the score; standard trading vocabulary (grey) is tracked but excluded. Substring matching only; the Python package adds jieba segmentation and negation handling. No text leaves your browser.

 Examples: speculator | value investor | lottery draw

Score a CSV file

Any UTF-8 CSV with a column of texts. Each row is scored and the table is returned with the scores appended. Processing runs locally in the browser.

Lexicon frequency scoring, in the tradition of dictionary-based indices (Baker–Bloom–Davis EPU; Loughran–McDonald). Full pipeline — segmentation, negation window, weakly-supervised classifier (CV AUC 0.867), daily index — in the Python package.
